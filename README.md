# IPL Match Analysis Dashboard

An interactive Power BI dashboard analyzing IPL (Indian Premier League) match data, providing insights into team performance, player statistics, and match trends across seasons.

## Overview

This project visualizes historical IPL match data to uncover patterns in team wins, player performance, venue statistics, and season-wise trends. The dashboard is built using Power BI for clear, interactive data exploration.

## Features

- Team-wise win/loss analysis
- Top performing players (batting & bowling stats)
- Season-wise match trends
- Venue and toss impact analysis
- Interactive filters for team, season, and player

## Tools Used

- Power BI – Dashboard creation and data visualization
- Excel/CSV – Data source and cleaning

## How to View

1. Download the `ipl dashboard.zip` file from this repository
2. Extract the ZIP file
3. Open the `.pbix` file in Power BI Desktop
